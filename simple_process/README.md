# Простой процесс

Пример демонстрирует самый простой процесс, который выводит всё что доступно ему в его контексте.

Код [примера](main.go) написан на языке `Go` и снабжен комментариями.

## Запуск и ожидаемый вывод

### Сборка из исходников

1. Скачать и установить go (использовался go 1.25.7)
2. Собрать пример
   ```bash
   cd simple_process/ # если вы в корневом каталоге проекта
   go build
   ```
3. Должен появиться исполняемый файл `simple_process` (`simple_process.exe` на Windows)

### Запуск примера

1. Войдите в каталог с исполняемым файлом (если вы еще не в нем)
2. Запустите пример
   ```bash
   ./simple_process "hello world" single args
   ```
3. Если все сделали правильно, должны увидеть примерно следующее:

   ```bash
   CWD: C:\SANDBOX\go\TUTORIAL\simple_process
   PID: 6072
   UID: -1

   argv [count: 4]:
   0:  C:\SANDBOX\go\TUTORIAL\simple_process\simple_process.exe
   1: hello world
   2: single
   3: args

   Environment Variable [count: 101]:
   000: ALLUSERSPROFILE=C:\ProgramData
   001: ANDROID_SDK_HOME=C:\Android
   ...
   ```

4. Вывод содержит:
   1. Путь к рабочей директории
   2. ID текущего процесса
   3. ID пользователя от которого работает процесс (в Windows всегда -1 так как в этой ОС немного другая система пользователей)
   4. Список аргументов командной строки `argv` содержит 4 аргумента:
      - полный путь к двоичному файлу приложения (в подавляющем большинстве случаев здесь (первым) будет именно он)
      - первый аргумент (`hello world`) который мы передали
      - второй и третий соответственно
   5. Список переменных окружения. Как можем видеть - в обычной настольной системе их довольно много
